`||agent:agent inspect||`コマンドは次のようになります:
```python
agent.inspect(AgentInspection.BLOCK, FORWARD)
```
コードで既に提供されている変数を使用してください: left = BLUE_GLAZED_TERRACOTTA, right = PINK_GLAZED_TERRACOTTA
